How It Works

User Flow:

  1. Initial Load: Mini panel loads fade duration from Tauri persistent storage
  2. Display: Slider position and value display show current duration
  3. Adjustment: User drags slider to new value (1-60 seconds range)
  4. Update: onChange handler immediately updates state and UI
  5. Persistence: New value saved to Tauri storage automatically
  6. Application: New shapes use the updated fade duration

Integration with Auto-Fade System:

The fade duration value is used by the Overlay component to determine when shapes should start fading. Each shape has a createdAt timestamp, and the fade animation starts when now - createdAt > fadeDuration * 0.8 (80% of lifetime).

// From src/components/Overlay.tsx const getFadeOpacity = useCallback((age: number, fadeDuration: number): number => { const fadeStart = fadeDuration * 0.8; // Start fading at 80% of lifetime const fadeEnd = fadeDuration; if (age < fadeStart) return 1.0; // Fully visible if (age >= fadeEnd) return 0.0; // Invisible // Smooth fade animation during last 20% const t = (age - fadeStart) / (fadeEnd - fadeStart); const opacity = 1 - Math.pow(1 - t, 3); return 1 - opacity; }, []);
