Feature #107: Tool-Specific Cursor Changes Test

Test that the cursor changes appearance based on the currently selected drawing tool.

Overview

Feature #107 enhances the user experience by changing the cursor appearance to match the selected drawing tool. This provides immediate visual feedback about which tool is active and what action the cursor will perform.

Previously, the cursor simply changed to "crosshair" when drawing mode was active. Now, each tool has an appropriate cursor style.

Implementation Details

✓ getToolCursor() Function (src/components/Overlay.tsx)

Added a helper function that returns the appropriate cursor based on the current tool and drawing mode:

const getToolCursor = useCallback((tool: ToolType | null, drawingMode: boolean): string => {
  // If not in drawing mode or no tool selected, use default cursor
  if (!drawingMode || !tool) {
    return "default";
  }

  // Return tool-specific cursor
  switch (tool) {
    case ToolType.ARROW:
      return "crosshair"; // Precision cursor for arrow drawing
    case ToolType.CIRCLE:
      return "crosshair"; // Precision cursor for circle drawing
    case ToolType.BOX:
      return "crosshair"; // Precision cursor for box drawing
    case ToolType.FREEHAND:
      return "crosshair"; // Standard drawing cursor for freehand
    case ToolType.HIGHLIGHTER:
      return "crosshair"; // Standard drawing cursor for highlighter
    case ToolType.TEXT:
      return "text"; // Text cursor (I-beam) for text input
    default:
      return "crosshair";
  }
}, []);

✓ Cursor Style Application

The overlay's cursor style is now set dynamically based on the current tool:

cursor: getToolCursor(currentTool, isDrawingMode)

This replaces the previous implementation that only checked isDrawingMode.

Why These Cursors Were Chosen

Cursor Type Tools Using It Rationale
crosshair Arrow, Circle, Box, Freehand, Highlighter The crosshair cursor (+) provides precision targeting, which is ideal for all drawing tools. It helps users accurately place the start and end points of shapes.
text Text The text cursor (I-beam) is the universal standard for text input. It clearly indicates to users that clicking will allow them to type text.
default No tool / Drawing mode off The default pointer cursor indicates normal interaction and that drawing is not active.
