Building Installers

Build for Current Platform

Command: 
npm run tauri build

This will:

  1. Build the frontend (npm run build)
  2. Compile the Rust backend
  3. Generate installers for the current platform

Code Signing (Optional)

Windows

To sign Windows installers and avoid SmartScreen warnings:

  1. Obtain code signing certificate (DigiCert, Sectigo, etc.)
  2. Install certificate on Windows build machine
  3. Update tauri.conf.json: 
    "windows": {
  "certificateThumbprint": "YOUR_CERT_THUMBPRINT",
  "digestAlgorithm": "sha256",
  "timestampUrl": "http://timestamp.digicert.com"
}
  4. Build - installer will be automatically signed

macOS

To sign macOS apps for distribution:

  1. Enroll in Apple Developer Program ($99/year)
  2. Get Developer ID certificate from Apple
  3. Update tauri.conf.json: 
    "macOS": {
  "signingIdentity": "Developer ID Application: Your Name (TEAM_ID)",
  "providerShortName": "TEAM_ID",
  "entitlements": "path/to/entitlements.plist"
}
  4. Build - app will be signed and notarized
