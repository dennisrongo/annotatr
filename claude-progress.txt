## Implementation Notes

**Freehand Tool:**
- Tracks points continuously during mouse drag
- Renders smooth path using `lineTo()` between consecutive points
- Minimum 2 points required to create shape
- Real-time preview shows path as it's drawn

**Highlighter Tool:**
- Same point-tracking mechanism as freehand
- Uses `globalAlpha = 0.3` for semi-transparent effect
- Doubles the line thickness for broader stroke
- Must reset `globalAlpha` to 1.0 after rendering

**Text Tool:**
- Uses inline input field instead of `prompt()` for better UX
- Input field positioned exactly at click location
- Enter submits, Escape cancels, blur auto-submits
- Default font size: 24px
- Text color follows current color selection

**All Tools:**
- Integrated with existing shape system
- Support undo/clear operations
- Persist in shapesRef until overlay dismissed
- Redrawn on canvas resize
- Tool indicator shows active tool in top-left corner
