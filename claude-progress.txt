## Implementation Notes

**Text Input Flow:**
1. User selects Text tool from MiniPanel
2. `activate_tool_hotkey` shows overlay and enables drawing mode
3. User clicks anywhere on overlay
4. Click coordinates captured and stored in `textInputPosition`
5. Input field rendered at click position and auto-focused
6. User types text and presses Enter (or clicks away)
7. Text shape created and added to `shapesRef`
8. Input field hidden, text rendered on canvas
9. Text shape persists until overlay is dismissed

**Key Improvements:**
- MiniPanel now properly enables drawing mode when selecting tools
- Text input uses inline field instead of browser prompt for better UX
- Auto-focus allows immediate typing without extra clicks
- Blur handling provides natural "click away to finish" interaction
- Escape key provides quick cancel without creating shape
