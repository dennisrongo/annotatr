### Technical Architecture Summary

**Storage Flow:**
```
Frontend (TypeScript)          Backend (Rust)
--------------------          --------------
saveSetting()         →        save_settings()
  invoke('save_settings')      StoreBuilder::new()
                                store.set()
                                store.save()

loadSettings()        →        load_settings()
  invoke('load_settings')      StoreBuilder::new()
                                store.as_json()
```

**Persistence:**
- Settings stored in: `settings.json` (app data directory)
- Managed by: Tauri's store plugin (tauri-plugin-store)
- Cross-platform: Works on Windows, macOS, Linux
- Format: JSON file with key-value pairs
