Key Architectural Decision

✓ CORRECT: Use Tauri 2's Unified Global Shortcut Plugin

The application uses tauri-plugin-global-shortcut v2, which provides a single, unified API for registering global hotkeys across Windows, macOS, and Linux. This is the industry-standard approach for cross-platform desktop applications.

Benefits:

  • Single codebase for all platforms
  • Platform differences handled by the plugin
  • Automatic updates when Tauri adds platform support
  • Reduced maintenance burden
  • Consistent behavior across platforms

Alternative (NOT used): Manual platform-specific code using Win32 API on Windows, Carbon/CGEvent on macOS, and X11/Wayland directly on Linux. This would require three separate code paths, increasing complexity and bug surface area.

Hotkey String Parser (lines 618-689)

fn parse_hotkey_string(s: &str) -> Result { let mut modifiers = Vec::new(); let mut key = None; for part in s.split('+') { match part.to_uppercase().as_str() { "CTRL" | "CONTROL" => modifiers.push(Modifier::Control), "SHIFT" => modifiers.push(Modifier::Shift), "ALT" => modifiers.push(Modifier::Alt), "META" | "CMD" | "SUPER" | "WIN" => modifiers.push(Modifier::Super), _ => key = Some(parse_key_string(part)?), } } Ok(Shortcut::new(modifiers, key?)) }
