Windows Optimizations

Windows Direct Composition & Hardware Acceleration
  • Direct Composition: Uses DirectX 11/12 for transparent window composition
  • ANGLE Backend: WebGL uses DirectX backend for better performance
  • GPU Rasterization: Canvas rendering accelerated by GPU
  • Zero-Copy Video: Reduced memory overhead for video frames
  • Raw Input: Bypasses Windows input processing for lower latency
  • Heap Compaction: Automatic memory defragmentation

Performance Tips:

  • Enable "Hardware-accelerated GPU scheduling" in Windows Settings
  • Set graphics preference to "High performance" for Annotatr
  • Use Game Mode when recording to prioritize app resources
  • Close unnecessary background apps to free system resources

macOS Optimizations

macOS Core Animation & Metal Acceleration
  • Core Animation: Hardware-accelerated window composition
  • Metal Backend: WebGL uses Metal for optimal GPU performance
  • Layer-Backed Views: Efficient view rendering for transparency
  • Async Frame Decoding: Parallel processing for better responsiveness
  • Pressure Sensitivity: Support for graphics tablets (Wacom, etc.)
  • Memory Pressure Monitoring: Automatic resource management

Performance Tips:

  • Close graphics-intensive apps (Photoshop, Final Cut) when recording
  • Use "Reduce Transparency" in Accessibility if performance is poor on older Macs
  • Ensure "Automatic graphics switching" is enabled in Energy Saver
  • Monitor CPU/GPU usage with Activity Monitor

Linux Optimizations

Linux OpenGL & Compositor Integration
  • OpenGL Backend: Native OpenGL acceleration for WebGL
  • VAAPI Acceleration: Hardware video decoding on Intel/AMD GPUs
  • X11 Composite: Transparent window support via X Composite
  • Wayland Support: Native Wayland rendering for modern distros
  • Tablet Support: XInput/Wacom tablet integration
  • Lazy Allocation: On-demand memory allocation for efficiency

Performance Tips:

  • Use NVIDIA/AMD proprietary drivers for best GPU performance
  • Enable compositing in KDE/GNOME for smooth transparency
  • Set CPU governor to "performance" mode for reduced latency
  • Use "PICMIP=0" for sharper text on NVIDIA cards
