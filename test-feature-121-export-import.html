Feature Overview

Category: Settings and Persistence

Description: Allow users to export and import settings configuration

Status: ✓ IMPLEMENTED AND PASSING

What is Export/Import?

Export/Import functionality allows users to:

  • Export: Save their current settings to a JSON file for backup or sharing
  • Import: Load settings from a JSON file to restore or apply a configuration

Benefits:

  • Backup: Users can backup their settings before making changes
  • Sharing: Users can share their settings with others
  • Migration: Users can transfer settings between computers
  • Restoration: Users can restore settings after a reset
  • Configuration: IT administrators can pre-configure settings for users

1. Backend Functions (src/lib/storage.ts)

Export Function:

/**
 * Export settings to a JSON file
 * Creates a downloadable JSON file with current settings
 */
export async function exportSettings(): Promise<void> {
  // Load current settings
  const settings = await loadSettings();

  // Create JSON string with pretty formatting
  const json = JSON.stringify(settings, null, 2);

  // Create blob with JSON data
  const blob = new Blob([json], { type: 'application/json' });

  // Create temporary URL
  const url = URL.createObjectURL(blob);

  // Create temporary anchor to trigger download
  const anchor = document.createElement('a');
  anchor.href = url;
  anchor.download = `annotatr-settings-${date}.json`;

  // Trigger download
  document.body.appendChild(anchor);
  anchor.click();

  // Cleanup
  document.body.removeChild(anchor);
  URL.revokeObjectURL(url);
}
Export Behavior:
  • Filename format: annotatr-settings-YYYY-MM-DD.json
  • Content: Complete settings object with all keys
  • Format: Pretty-printed JSON (2-space indentation)
  • Trigger: User clicks "Export" button
  • Browser: Shows standard save dialog

Import Function:

/**
 * Import settings from a JSON file
 * Reads a JSON file and applies the settings
 */
export async function importSettings(file: File): Promise<Settings> {
  // Read file as text
  const text = await file.text();

  // Parse JSON
  const imported = JSON.parse(text);

  // Validate structure
  const validated = validateImportedSettings(imported);

  // Save all settings to storage
  await saveSettings(validated);

  return validated;
}
Import Behavior:
  • Trigger: User clicks "Import" button
  • File picker: Opens standard file selection dialog
  • File type: Accepts only .json files
  • Validation: Checks structure and value ranges
  • Merge: Merges with defaults for missing keys
  • Confirmation: Shows success/error alert

Validation Function:

/**
 * Validate imported settings structure
 */
function validateImportedSettings(imported: unknown): Settings {
  // Must be an object
  if (!imported || typeof imported !== 'object') {
    throw new Error('Invalid format: expected an object');
  }

  const settings = imported as Record<string, unknown>;

  // Validate hotkeys object exists
  if (!settings.hotkeys || typeof settings.hotkeys !== 'object') {
    throw new Error('Missing or invalid hotkeys');
  }

  // Validate colors object exists
  if (!settings.colors || typeof settings.colors !== 'object') {
    throw new Error('Missing or invalid colors');
  }

  // Validate line thickness range (1-50)
  if (typeof settings.lineThickness !== 'number' ||
      settings.lineThickness < 1 || settings.lineThickness > 50) {
    throw new Error('lineThickness must be between 1 and 50');
  }

  // Validate font size range (8-72)
  if (typeof settings.fontSize !== 'number' ||
      settings.fontSize < 8 || settings.fontSize > 72) {
    throw new Error('fontSize must be between 8 and 72');
  }

  // Validate fade duration range (1-60)
  if (typeof settings.fadeDuration !== 'number' ||
      settings.fadeDuration < 1 || settings.fadeDuration > 60) {
    throw new Error('fadeDuration must be between 1 and 60');
  }

  // Merge with defaults
  return {
    ...DEFAULT_SETTINGS,
    ...(settings as Settings),
  };
}

2. Frontend Handlers (src/components/MiniPanel.tsx)

Export Handler:

const handleExportSettings = async () => {
  try {
    await exportSettings();
    console.log("Settings exported successfully");
    alert("Settings exported successfully!");
  } catch (error) {
    console.error("Failed to export settings:", error);
    alert("Failed to export settings. Please try again.");
  }
};

Import Handler:

const handleImportSettings = async () => {
  // Create file input element
  const input = document.createElement("input");
  input.type = "file";
  input.accept = "application/json";

  // Handle file selection
  input.onchange = async (e) => {
    const file = (e.target as HTMLInputElement).files?.[0];
    if (!file) return;

    try {
      // Import and validate
      const imported = await importSettings(file);

      // Update all state variables
      setCurrentColorForTool(imported.colors);
      setSelectedColor(imported.colors.arrow);
      setHotkeys(imported.hotkeys);
      setLineThickness(imported.lineThickness);
      setFontSize(imported.fontSize);
      setFadeDuration(imported.fadeDuration);

      // Clear conflicts and re-register hotkeys
      setHotkeyConflicts({});
      await invoke("register_hotkeys", { hotkeyConfig: imported });

      alert("Settings imported successfully!");
      setShowSettingsModal(false);
    } catch (error) {
      alert(`Failed to import: ${error.message}`);
    }
  };

  // Trigger file picker
  input.click();
};
