How It Works

User Flow:

  1. Initial Load: Mini panel loads line thickness from Tauri persistent storage
  2. Display: Slider position and value display show current thickness
  3. Adjustment: User drags slider to new value (1-50px range)
  4. Update: onChange handler immediately updates state and UI
  5. Persistence: New value saved to Tauri storage automatically
  6. Next Launch: App loads saved thickness value on startup

Integration with Drawing:

The line thickness value is used when creating shapes in the overlay. Each shape stores its own lineThickness property, which is set from the current value when the shape is created.

// From types/shapes.ts export interface BaseShape { id: string; tool: ToolType; color: string; lineThickness: number; // ← Line thickness stored per shape createdAt: number; }
