## Technical Notes - Feature #132

### Detection Algorithm

The alignment detection uses simple distance calculations:

```typescript
// Center detection
const centerX = canvas.width / (2 * devicePixelRatio);
const centerY = canvas.height / (2 * devicePixelRatio);
horizontal = Math.abs(y - centerY) < SNAP_THRESHOLD;
vertical = Math.abs(x - centerX) < SNAP_THRESHOLD;

// Edge detection
left = Math.abs(x) < SNAP_THRESHOLD;
right = Math.abs(x - width) < SNAP_THRESHOLD;
top = Math.abs(y) < SNAP_THRESHOLD;
bottom = Math.abs(y - height) < SNAP_THRESHOLD;
```

### Snapping Behavior

When alignment is detected, the position automatically snaps:

```typescript
let snappedX = currentX;
let snappedY = currentY;

if (centerAlign.horizontal) {
  snappedY = centerY;
}
if (centerAlign.vertical) {
  snappedX = centerX;
}
```

### Visual Guide Rendering

Guides use standard Canvas 2D API:

```typescript
ctx.strokeStyle = "#00BFFF";
ctx.setLineDash([5, 5]);  // Dashed
ctx.globalAlpha = 0.7;     // Semi-transparent

// Draw line
ctx.beginPath();
ctx.moveTo(0, guideY);
ctx.lineTo(width, guideY);
ctx.stroke();

// Draw snap point
ctx.beginPath();
ctx.arc(snappedX, snappedY, 4, 0, Math.PI * 2);
ctx.fill();
```
