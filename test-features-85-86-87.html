Feature #87: Settings saved to local storage

Description: Implement saving settings to Tauri's local storage when save button is clicked

Implementation:

  • Backend: Rust save_settings command in lib.rs (lines 53-75)
  • Uses Tauri's store plugin for cross-platform persistent storage
  • Stores data in settings.json file
  • Frontend: saveSettings() function in storage.ts (lines 73-84)
  • Serializes settings data as JSON
  • Emits "settings_updated" event after save
  • Integration: Save button handler calls saveSettings() with all current settings
