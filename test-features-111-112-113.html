How It Works

The z-ordering system ensures that newly created shapes render on top of older shapes:

  1. Shape Storage: All shapes are stored in shapesRef.current array in order of creation
  2. Rendering Order: The redrawShapes() function iterates through the array and draws each shape sequentially
  3. Canvas Layering: In HTML5 Canvas, shapes drawn later appear on top (higher z-index) than shapes drawn earlier

2. Shape Rendering (src/lib/drawing.ts)

export function redrawShapes(
  ctx: CanvasRenderingContext2D,
  shapes: Shape[],
  opacities: Record = {}
): void {
  // Clear canvas
  ctx.clearRect(0, 0, ctx.canvas.width, ctx.canvas.height);

  // Draw shapes in array order (older first, newer last)
  shapes.forEach((shape) => {
    const opacity = opacities[shape.id] ?? 1.0;
    drawShape(ctx, shape, opacity);
  });
}

Backend Implementation (src-tauri/src/lib.rs)

#[tauri::command]
fn set_drawing_mode(app: AppHandle, enabled: bool) -> Result<(), String> {
    // Show or hide the overlay window based on drawing mode
    if enabled {
        // Show overlay and make it capture input
        show_overlay(app.clone())?;
    } else {
        // Hide overlay
        hide_overlay(app.clone())?;
    }

    // Emit event to notify frontend
    app.emit("drawing-mode-changed", enabled)
        .map_err(|e| format!("Failed to emit drawing-mode-changed event: {}", e))?;

    Ok(())
}

How It Works

Drawing mode deactivation disables shape drawing and cleans up state:

  1. Backend Call: Invokes Tauri command set_drawing_mode with enabled: false
  2. State Cleanup: Resets all drawing-related state to default values
  3. Overlay Hidden: Overlay window hides or stops capturing input
  4. Cursor Reset: Cursor changes back to default pointer
  5. Visual Indicator: Tool indicator disappears or shows inactive state

Escape Key Handler (Overlay.tsx, lines 612-640)

const handleKeyDown = async (event: KeyboardEvent) => {
  if (event.key === "Escape") {
    console.log("Escape key pressed - dismissing overlay");
    event.preventDefault();

    try {
      // Call dismiss_overlay command (deactivates drawing mode)
      await invoke("dismiss_overlay");

      // Clean up any active drawing state
      setDrawingState({
        isDrawing: false,
        currentTool: null,
        startX: 0,
        startY: 0,
        currentX: 0,
        currentY: 0,
        freehandPoints: [],
        textInput: "",
        textPosition: null,
      });
      setCurrentTool(null);
      setIsVisible(false);
    } catch (error) {
      console.error("Failed to dismiss overlay:", error);
    }
  }
};
