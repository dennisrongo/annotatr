How It Works

All three hotkeys use the existing hotkey infrastructure implemented in Features #56, #57, #58:

  1. App Startup: MiniPanel.tsx calls register_hotkeys() with default settings
  2. Registration: Backend iterates through all hotkeys in config and registers them with OS
  3. Trigger: When user presses Ctrl+Shift+F/T/H, the global shortcut is triggered
  4. Handler: activate_tool_hotkey() is called with the tool name
  5. Activation: Overlay is shown, drawing mode enabled, tool-selected event emitted
  6. UI Update: Overlay receives tool-selected event and updates current tool

#[tauri::command]
fn register_hotkeys(app: AppHandle, hotkey_config: serde_json::Value) -> Result<(), String> {
    let hotkeys_obj = hotkey_config["hotkeys"].as_object()?;
    for (tool_name, hotkey_str) in hotkeys_obj.iter() {
        let shortcut = parse_hotkey_string(hotkey_str)?;
        let app_handle = app.clone();
        let tool = tool_name.clone();
        app.global_shortcut().register(shortcut, move || {
            activate_tool_hotkey(app_handle.clone(), tool.clone()).ok();
        })?;
        println!("Registered hotkey '{}' for tool '{}'", hotkey_value, tool_name);
    }
    Ok(())
}
