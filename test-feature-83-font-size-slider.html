How It Works

User Flow:

  1. Initial Load: Mini panel loads font size from Tauri persistent storage
  2. Display: Slider position and value display show current font size
  3. Adjustment: User drags slider to new value (8-72pt range, 2pt increments)
  4. Update: onChange handler immediately updates state and UI
  5. Persistence: New value saved to Tauri storage automatically
  6. Next Launch: App loads saved font size on startup

Integration with Text Tool:

The font size value is used when creating text shapes in the overlay. Each text shape stores its own fontSize property, which is set from the current value when the text is created.

// From types/shapes.ts export interface TextShape extends BaseShape { tool: ToolType.TEXT; position: Point; text: string; fontSize: number; // ← Font size stored per text shape }
