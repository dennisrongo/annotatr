## Technical Notes

### Feature #124: Clear All Shapes

**Memory Management:**
Critical to clean up `shapeOpacities` for all cleared shapes. The implementation uses
the centralized `cleanupOpacityTracking()` function from Feature #120 to prevent
memory leaks.

**Keyboard Shortcut Choice:**
Ctrl+Shift+X was chosen because:
- X suggests "eXterminate" or "X out"
- Doesn't conflict with common shortcuts
- Easy to remember
- Cross-platform compatible

### Feature #131: Arrow Head Styles

**Arrow Head Geometry:**
- Head Length = lineThickness × 3
- Head Angle = ±30 degrees from shaft angle
- This ratio ensures heads are visible but not overwhelming

**Style Rendering:**
- FILLED: Uses `ctx.fill()` for solid color
- OPEN: Uses `ctx.stroke()` for outline only
- DOUBLE_HEADED: Two filled heads (one rotated 180°)

**Default Choice:**
FILLED style is the default because:
- Best visibility
- Most common in annotation tools
- Matches existing behavior (backward compatible)
