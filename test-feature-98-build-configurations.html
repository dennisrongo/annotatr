Platform Detection Code (src-tauri/src/utils.rs)

Compile-Time Platform Detection

// Detect the current platform at compile time pub fn get_platform() -> Platform { #[cfg(target_os = "windows")] { return Platform::Windows; } #[cfg(target_os = "macos")] { return Platform::Macos; } #[cfg(target_os = "linux")] { return Platform::Linux; } #[cfg(not(any( target_os = "windows", target_os = "macos", target_os = "linux" )))] { return Platform::Unknown; } }
Conditional Compilation: Rust's #[cfg(...)] attributes allow code to be compiled differently for each platform. The get_platform() function is evaluated at compile time, not runtime, for zero overhead.
