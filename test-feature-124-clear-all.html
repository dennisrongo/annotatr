Feature Description

Add ability to clear all shapes from the overlay at once using a keyboard shortcut. This allows users to quickly remove all annotations from the screen without having to undo each shape individually.

Function Implementation

const clearAllShapes = useCallback(() => {
  if (shapesRef.current.length === 0) {
    console.log("[Feature #124] No shapes to clear");
    return;
  }

  // Get all shape IDs before clearing
  const shapeIds = shapesRef.current.map(shape => shape.id);
  const shapeCount = shapesRef.current.length;

  // Clear all shapes
  shapesRef.current = [];

  // Feature #120: Clean up opacity tracking for all cleared shapes
  cleanupOpacityTracking(shapeIds);

  console.log(`[Feature #124] Cleared ${shapeCount} shape(s)`);
  console.log(`[Feature #124] Shape IDs: ${shapeIds.join(", ")}`);

  // Redraw the canvas (should now be empty)
  redrawAllShapes();
}, [cleanupOpacityTracking, redrawAllShapes]);

Key Behaviors

  • Clears All Shapes: Removes all shapes from shapesRef.current array
  • Memory Cleanup: Calls cleanupOpacityTracking() for all cleared shapes (Feature #120 integration)
  • Canvas Update: Calls redrawAllShapes() to refresh the canvas (should now be empty)
  • Graceful Handling: Logs message and returns early if no shapes exist
  • Console Logging: Provides detailed logging of cleared shapes for debugging
  • Works with All Tools: Clears all 6 tool types (arrow, circle, box, freehand, highlighter, text)
